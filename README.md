# Common-Tools
## 工具库
### 内容如下：
|类名|说明|
|---|----|
|Base64Util|Base64 编码工具|
|MD5Util|MD5 工具类|
|DesUtil|DES 对称加解密|
|RSAUtil|RSA 加解密工具|
|SHA1Util|SHA1 工具|
|SPUtil|SharedPreference 工具|
|BitmapUtil|Bitmap 工具|
|CloseableUtil|IO 关闭工具，避免多次try-catch|
|CollectionUtil|集合工具，判空，去重，打印之类|
|HandlerUtil|Handler 工具，回主线程工具|
|HexUtil|十六进制工具|
|Log|日志，有开关控制打不打印|
